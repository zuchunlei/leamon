java中的移位运算
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
<< , >>, >>>

<< 左移 。二进制形式向左移动，移动以后空位补0。右边用0补齐。
>> 右移。java中整数类型为有符号位的，该右移不考虑符号位，也就说最大位的符号位在右移过程中保存不变。左边用符号位补齐。
>>> 算术右移。该移动过程中，符号位跟着移动，移动以后空位补0。左边用0补齐。

java中的位操作运算
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
&（按位与）：二目运算符，同位上只有同时为1时才为1，其它情况均为0。例：0x10101010&0x01010101=0x00000000
|（按位或）：二目运算符，同位上只有同时为0时才为0，其它情况均为1。例：0x10101010|0x01010101=0x11111111
^（按位异或）：二目运算符，同位上的值不相同是为1，相同则为0。例：0x10101010^0x01010101=0x11111111
~（按位取反）：单目运算符，如果为1，则取0；如果为0，则取1。例：~0x10101010=0x01010101

java流中read与write读写单字节的情形
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
int read() 返回一个int类型的数值，其实质上是一个0-255的无符号字节数值。
一般的操作是这样子的  byte & 0xff
其中0xff是作为int类型来运算的。
